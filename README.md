# Harry Potter Character Explorer

## Description
This is an Angular application built for COMP 3133 Lab Test 2 (Option 2b – Harry Potter Theme).  
The application consumes the HP API and allows users to browse Harry Potter characters, filter them by house, and view detailed character information.

## Features Implemented
- Fetch all Harry Potter characters from public API
- Display character name, house, and image
- Filter characters by house using dropdown
- View character details on a separate page
- Angular Material UI components
- Angular HttpClient integration
- Reactive form dropdown
- Uses Angular control flow: @for, @if, @switch
- Uses Angular signals

## Technologies Used
- Angular
- TypeScript
- Angular Material
- HttpClient
- Reactive Forms
- HP API

## API Endpoints Used
- /api/characters
- /api/characters/house/:house
- /api/character/:id

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `ng serve`
4. Open `http://localhost:4200`
